---
title: Stack
---

核心用途：堆叠多个子元素，支持层级控制。

参数：
- `children`: 子元素数组（必填）

示例：

import CodeBlock from '@theme/CodeBlock';

<CodeBlock live language="jsx">
{`
function StackDemo() {
  return (
    <div style={{ position: 'relative', width: 200, height: 120, border: '1px solid #ddd' }}>
      <div style={{ position: 'absolute', left: 20, top: 20, width: 80, height: 40, background: '#1677ff' }} />
      <div style={{ position: 'absolute', left: 50, top: 40, width: 80, height: 40, background: '#52c41a', opacity: 0.8 }} />
    </div>
  );
}
`}
</CodeBlock>

最佳实践：
- 控制层级与透明度，避免可访问性问题。
