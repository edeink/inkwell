# 交互式计数器

这是一个展示 Inkwell 框架状态管理与组件通信的示例。

## 在线演示

```tsx mode:edit
/** @jsxImportSource @/utils/compiler */
<InteractiveCounterDemo />
```

## 功能特性

1.  **多组件类型支持**
    *   **Class Component**: 使用 `StatefulWidget` 实现的计数器按钮。
    *   **Functional Component**: 使用函数式组件实现的按钮。
    *   **Raw Widget**: 直接操作 RenderObject 的低级组件。

2.  **状态管理**
    *   展示了父子组件之间的通信。
    *   使用 `setState` 更新状态并触发重绘。

3.  **性能监控**
    *   实时显示 FPS 和节点数量。

## 代码结构

*   `src/demo/interactive-counter/widgets/interactive-counter-demo/index.tsx`: 主入口。
*   `src/demo/interactive-counter/widgets/class-button/index.tsx`: 类组件实现。
*   `src/demo/interactive-counter/widgets/functional-button/index.tsx`: 函数组件实现。
