# RoadMap

本项目是基于 Vibe + Spec 的方式（且 Vibe 的成分更大），所以此文件并不是严格的 RoadMap，而是笔者正在或准备做哪一些更有趣的建设。仅供了解。

## Building

- [ ] 增量更新（优化性能，规范生命周期）
  - [ ] 脏区域与局部重绘（减少全量 paint）
  - [ ] 布局/绘制阶段性能基准（为优化提供可量化指标）
- [ ] 文本排版能力补强（换行、截断、基线、富文本一致性）
  - [ ] 资源与缓存体系（图片解码/缓存、字体加载、复用策略）
- [ ] 超级有趣的组件库（生态）

## Prepare

- [ ] Build 优化，预编译
- [ ] 函数式编程（为了节省内存）
- [ ] Composite 多图层
- [ ] 插件系统（扩展能力）

## Maybe

- [ ] 渲染器扩展点标准化（Canvas2D / Pixi / Konva 行为一致）
- [ ] WebGL & Shader
- [ ] 支持图文编辑器（进一步验证能力）
  - [ ] 文本编辑基础设施（选区、光标、IME、撤销/重做、段落布局）
  - [ ] 画布交互系统补强（框选、多选、对齐线、吸附、缩放体验）
