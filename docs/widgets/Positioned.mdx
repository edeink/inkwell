---
title: Positioned
---

核心用途：在 `Stack` 中通过绝对定位放置单个子元素，支持 `left/top/right/bottom/width/height`。

## 如何引入
- 项目代码：`import { Stack, Positioned, Container, Text } from '@/core'`
- 文档/示例：在代码块首行添加 `/** @jsxImportSource @/utils/compiler */`

## 示例

### 基础定位

```tsx
/** @jsxImportSource @/utils/compiler */
<Stack>
  <Positioned left={12} top={12} width={80} height={40}>
    <Container color="#e6f4ff" />
  </Positioned>
</Stack>
```

## 属性

| 属性名 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| `left` | `number` | 否 | 无 | 左侧偏移 |
| `top` | `number` | 否 | 无 | 顶部偏移 |
| `right` | `number` | 否 | 无 | 右侧偏移 |
| `bottom` | `number` | 否 | 无 | 底部偏移 |
| `width` | `number` | 否 | 无 | 固定宽度（与 `left/right` 同时存在时由二者决定） |
| `height` | `number` | 否 | 无 | 固定高度（与 `top/bottom` 同时存在时由二者决定） |

- children 支持类型：`Single`

## Tips
- Positioned 仅在 `Stack` 中生效；实际绘制位置由 `Stack` 根据父尺寸计算。
- 当同时设置 `left/right` 或 `top/bottom` 时，会根据父尺寸与边距进行拉伸计算（参考 `src/core/positioned.ts:89-99`、`102-135`）。
