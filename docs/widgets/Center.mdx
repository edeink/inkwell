---
title: Center
---

核心用途：将子元素在父容器内水平与垂直方向同时居中。

## 如何引入
- 项目代码：`import { Center, Container, Text } from '@/core'`
- 文档/示例：在代码块首行添加 `/** @jsxImportSource @/utils/compiler */`

常见属性与用法
- Center 仅接收一个子元素；如需多个元素，请将子元素包装在 `Row` 或 `Column` 中。
- 配合 `Container` 的 `width/height/padding/border` 控制可视区域与边界。

## 示例

### 最简基础用法

```esx
/** @jsxImportSource @/utils/compiler */
<Container width={300} height={120} color="#e6f4ff">
  <Center>
    <Text text="居中元素" />
  </Center>
</Container>
```

## 属性

| 属性名 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| `flex` | 布局扩展属性（权重与适应方式） | `&#123; flex?: number; fit?: 'tight' | 'loose' &#125;` | 无 |

- children 支持类型：`Single`

## Tips

- 在需要固定居中效果时优先使用 Center；避免与绝对定位混用导致覆盖关系复杂。
