---
title: Select
---

核心用途：下拉选择器。支持非受控（`defaultValue`）与受控（`value`）两种模式，支持禁用选项与滚动列表。

## 如何引入

- 项目代码：`import { Select } from '@/comp'`
- 文档/示例：在代码块首行添加 `/** @jsxImportSource @/utils/compiler */`

## 示例

### 基础用法

```tsx mode:edit
/** @jsxImportSource @/utils/compiler */
<Stack allowOverflowPositioned={true}>
  <Container width={520} height={200} color="#ffffff" border={{ width: 1, color: '#e5e7eb' }} borderRadius={8} />
  <Positioned left={16} top={16}>
    <Column spacing={12} crossAxisAlignment={CrossAxisAlignment.Start}>
      <Select
        width={240}
        placeholder="请选择城市"
        options={[
          { label: '北京', value: 'beijing' },
          { label: '上海', value: 'shanghai' },
          { label: '深圳', value: 'shenzhen' },
          { label: '杭州（禁用）', value: 'hangzhou', disabled: true },
        ]}
        onChange={(v) => message.success(`已选择：${v}`)}
      />
      <Select
        width={240}
        disabled={true}
        placeholder="已禁用"
        options={[{ label: '选项', value: 'x' }]}
      />
    </Column>
  </Positioned>
</Stack>
```

## 属性

| 属性名 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| `width` | 组件宽度 | `number` | 无 |
| `options` | 选项列表 | `ReadonlyArray<{ label: string; value: T; disabled?: boolean; key?: string }>` | 无 |
| `value` | 受控值 | `T \| null` | 无 |
| `defaultValue` | 非受控默认值 | `T \| null` | 无 |
| `placeholder` | 未选择时文案 | `string` | `'请选择'` |
| `disabled` | 禁用 | `boolean` | `false` |
| `size` | 尺寸 | `'small' \| 'middle' \| 'large'` | `'middle'` |
| `theme` | 主题（可选） | `ThemePalette` | 当前主题模式 |
| `onChange` | 变化回调 | `(value: T) => void` | 无 |

## Tips

- 下拉面板在 pointer 离开时会自动关闭；如需更复杂的关闭策略，可在外层自行做事件隔离。
