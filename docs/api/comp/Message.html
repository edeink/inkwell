On this page

核心用途：轻量全局提示。通过 message.* 触发消息，组件内部会基于 runtime 的 overlay 机制进行渲染。

如何引入 ​

  • 项目代码：import { message } from '@/comp'
  • 文档/示例：在代码块首行添加 /** @jsxImportSource @/utils/compiler */

示例 ​

基础用法 ​

/** @jsxImportSource @/utils/compiler */
(() => {
  const theme = getCurrentTheme();
  const height = 32;
  return (
    <Stack allowOverflowPositioned={true}>
      <Container
        width={520}
        height={180}
        border={{ width: 1, color: theme.border.secondary }}
        borderRadius={8}
        color={theme.background.container}
      />
      <Positioned left={16} top={16}>
        <Row spacing={8} mainAxisSize="min">
          <Button
            theme={theme}
            btnType="primary"
            onClick={() => message.success('操作成功', { duration: 1800 })}
          >
            <Text
              text="成功"
              fontSize={14}
              lineHeight={height}
              color={theme.text.inverse}
              textAlignVertical={TextAlignVertical.Center}
              pointerEvent="none"
            />
          </Button>
          <Button theme={theme} onClick={() => message.info('这是一条提示')}>
            <Text
              text="提示"
              fontSize={14}
              lineHeight={height}
              color={theme.text.primary}
              textAlignVertical={TextAlignVertical.Center}
              pointerEvent="none"
            />
          </Button>
          <Button theme={theme} danger={true} onClick={() => message.error('发生错误')}>
            <Text
              text="错误"
              fontSize={14}
              lineHeight={height}
              color={theme.danger}
              textAlignVertical={TextAlignVertical.Center}
              pointerEvent="none"
            />
          </Button>
        </Row>
      </Positioned>
    </Stack>
  );
})()

API ​

message ​

方法说明类型
message.info信息提示(content: string, options?: MessageOptions) => void
message.success成功提示(content: string, options?: MessageOptions) => void
message.warning警告提示(content: string, options?: MessageOptions) => void
message.error错误提示(content: string, options?: MessageOptions) => void

MessageOptions ​

字段说明类型默认值
duration自动关闭时间（毫秒）numberinfo: 2000 / success: 2000 / warning: 2500 / error: 3000
placement弹出位置'topRight' | 'center''topRight'
