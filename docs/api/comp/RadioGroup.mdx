---
title: RadioGroup
---

核心用途：单选组。以 `options` 描述可选项，内部维护非受控状态（`defaultValue`），也支持受控（`value`）。

## 如何引入

- 项目代码：`import { RadioGroup } from '@/comp'`
- 文档/示例：在代码块首行添加 `/** @jsxImportSource @/utils/compiler */`

## 示例

### 非受控（默认值）

```tsx mode:edit
/** @jsxImportSource @/utils/compiler */
<Stack allowOverflowPositioned={true}>
  <Container width={520} height={180} color="#ffffff" border={{ width: 1, color: '#e5e7eb' }} borderRadius={8} />
  <Positioned left={16} top={16}>
    <RadioGroup
      direction="vertical"
      options={[
        { label: '公开', value: 'public' },
        { label: '仅自己可见', value: 'private' },
        { label: '归档', value: 'archive', disabled: true },
      ]}
      defaultValue="public"
      onChange={(v) => message.info(`当前选择：${v}`)}
    />
  </Positioned>
</Stack>
```

## 属性

| 属性名 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| `options` | 选项列表 | `ReadonlyArray<{ label: string; value: T; disabled?: boolean; key?: string }>` | 无 |
| `value` | 受控值 | `T` | 无 |
| `defaultValue` | 非受控默认值 | `T` | 无 |
| `disabled` | 整组禁用 | `boolean` | `false` |
| `direction` | 排列方向 | `'horizontal' \| 'vertical'` | `'horizontal'` |
| `theme` | 主题（可选） | `ThemePalette` | 当前主题模式 |
| `onChange` | 变化回调 | `(value: T) => void` | 无 |

## Tips

- 受控模式下传入 `value` 时，需要在外部更新 `value` 才会反映 UI 变化。
