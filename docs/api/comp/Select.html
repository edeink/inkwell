On this page

核心用途：下拉选择器。支持非受控（defaultValue）与受控（value）两种模式，支持禁用选项与滚动列表。

如何引入 ​

  • 项目代码：import { Select } from '@/comp'
  • 文档/示例：在代码块首行添加 /** @jsxImportSource @/utils/compiler */

示例 ​

基础用法 ​

/** @jsxImportSource @/utils/compiler */
<Stack allowOverflowPositioned={true}>
  <Container
    width={520}
    height={200}
    color={getCurrentTheme().background.container}
    border={{ width: 1, color: getCurrentTheme().border.secondary }}
    borderRadius={8}
  />
  <Positioned left={16} top={16}>
    <Column spacing={12} crossAxisAlignment={CrossAxisAlignment.Start}>
      <Select
        width={240}
        placeholder="请选择城市"
        options={[
          { label: '北京', value: 'beijing' },
          { label: '上海', value: 'shanghai' },
          { label: '深圳', value: 'shenzhen' },
          { label: '杭州（禁用）', value: 'hangzhou', disabled: true },
        ]}
        onChange={(v) => message.success(`已选择：${v}`)}
      />
      <Select
        width={240}
        disabled={true}
        placeholder="已禁用"
        options={[{ label: '选项', value: 'x' }]}
      />
    </Column>
  </Positioned>
</Stack>

属性 ​

属性名说明类型默认值
width组件宽度number无
options选项列表ReadonlyArray<{ label: string; value: T; disabled?: boolean; key?: string }>无
value受控值T | null无
defaultValue非受控默认值T | null无
placeholder未选择时文案string'请选择'
disabled禁用booleanfalse
size尺寸'small' | 'middle' | 'large''middle'
theme主题（可选）ThemePalette当前主题模式
onChange变化回调(value: T) => void无
onOpen打开回调() => void无
onClose关闭回调() => void无

Tips ​

  • 下拉面板通过 Overlay 渲染；点击触发器切换打开/关闭，点击遮罩关闭。
