---
title: Button
---

核心用途：通用按钮组件，支持类型（default/primary/text/link 等）、尺寸、禁用与加载态。

## 如何引入

- 项目代码：`import { Button } from '@/comp'`
- 文档/示例：在代码块首行添加 `/** @jsxImportSource @/utils/compiler */`

## 示例

### 基础用法

```tsx mode:edit
/** @jsxImportSource @/utils/compiler */
<Row spacing={12} mainAxisSize="min">
  <Button btnType="default" onClick={() => console.log('点击默认按钮')}>
    <Text text="默认" pointerEvent="none" />
  </Button>
  <Button btnType="primary" onClick={() => console.log('点击主按钮')}>
    <Text text="主按钮" color="#ffffff" pointerEvent="none" />
  </Button>
  <Button btnType="text" onClick={() => console.log('点击文本按钮')}>
    <Text text="文本" pointerEvent="none" />
  </Button>
</Row>
```

### 禁用与加载态

```tsx mode:edit
/** @jsxImportSource @/utils/compiler */
<Row spacing={12} mainAxisSize="min">
  <Button disabled={true}>
    <Text text="禁用" pointerEvent="none" />
  </Button>
  <Button loading={true} btnType="primary">
    <Text text="提交" color="#ffffff" pointerEvent="none" />
  </Button>
</Row>
```

## 属性

| 属性名 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| `btnType` | 按钮类型 | `'default' \| 'primary' \| 'dashed' \| 'text' \| 'link'` | `'default'` |
| `size` | 组件尺寸 | `'small' \| 'middle' \| 'large'` | `'middle'` |
| `danger` | 危险按钮语义（影响主色） | `boolean` | `false` |
| `disabled` | 禁用 | `boolean` | `false` |
| `loading` | 加载态（显示“加载中”） | `boolean` | `false` |
| `width` | 固定宽度 | `number` | 无 |
| `theme` | 主题（可选） | `ThemePalette` | 当前主题模式 |
| `onClick` | 点击回调 | `(e: InkwellEvent) => void` | 无 |

- children 支持类型：`Multi`

## Tips

- 文本类子元素建议设置 `pointerEvent="none"`，避免命中测试落在子节点导致 hover/active 行为不一致。
