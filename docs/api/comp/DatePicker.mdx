---
title: DatePicker
---

核心用途：日期选择器，支持受控/非受控模式与弹出日历面板。

## 如何引入

- 项目代码：`import { DatePicker } from '@/comp'`
- 文档/示例：在代码块首行添加 `/** @jsxImportSource @/utils/compiler */`

## 示例

### 基础用法（非受控）

```tsx mode:edit
/** @jsxImportSource @/utils/compiler */
<DatePicker width={220} placeholder="请选择日期" onChange={(d) => console.log('选择日期', d)} />
```

## 属性

| 属性名 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| `width` | 触发器宽度 | `number` | 必填 |
| `size` | 组件尺寸 | `'small' \| 'middle' \| 'large'` | `'middle'` |
| `value` | 受控值 | `Date \| null` | 无 |
| `defaultValue` | 非受控初始值 | `Date \| null` | 无 |
| `placeholder` | 占位文本 | `string` | `'请选择日期'` |
| `disabled` | 禁用 | `boolean` | `false` |
| `theme` | 主题（可选） | `ThemePalette` | 当前主题模式 |
| `onChange` | 选择日期回调 | `(date: Date) => void` | 无 |

- children 支持类型：`None`

## Tips

- `value` 传入时启用受控模式；不传 `value` 则使用内部状态（`defaultValue` 仅初始化生效）。
