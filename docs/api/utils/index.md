title: 工具函数
---

本目录收敛两类“工具”：

- **Widget 树工具**：基于 `src/core` 的树结构与运行时能力（例如 `findWidget`）
- **通用工具**：位于 `src/utils` 的编译器/运行时辅助（例如 JSX 编译、localStorage 封装）

## 目录

| 文档 | 说明 |
| --- | --- |
| [选择器](/docs/api/utils/selector) | 在 Widget 树中按选择器查找组件 |
| [JSX 编译器](/docs/api/utils/jsx-compiler) | `compileElement/compileTemplate` 把 JSX 转为组件数据 |
